# InterviewAce

A Spring Boot backend project for interview preparation.

## Features

- User Registration
- User Login
- JWT Authentication
- Spring Security
- PostgreSQL Database

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- JWT
- PostgreSQL
- Maven

## Setup

1. Clone repository
2. Configure application.properties
3. Run PostgreSQL
4. Start application

## API Endpoints

POST /api/auth/register
POST /api/auth/login
